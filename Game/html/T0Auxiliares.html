Desenhar Menu

drawMenu :: ([[Tipos]], [Picture]) -> (Int, Int) -> [Picture] #

Através do Translate e do uso da lista de listas, é feito um distanciamento entre as Pictures. Desta forma, é desenhado primeiro horizontalmente, portanto, começamos por usar o Translate para que, a cada elemento de uma lista seja feito o distanciamento em x.

  • Os inteiros em Translate variam de acordo com aquilo que for o objetivo de reproduzir na tela.

drawMenu2 :: ([Tipos], [Picture]) -> Int -> [Picture] #

À semelhança da função anterior, aqui também é feito um distanciamento de Pictures, mas verticalmente.

Isto é, vai desenhando as colunas (verticalmente), usando também o Translate mas em y.

  • Os inteiros em Translate variam de acordo com aquilo que for o objetivo de reproduzir na tela.

opcoes :: (Tipos, [Picture]) -> Picture #

Esta função utiliza os elementos da lista para atribuir correspondência a cada Imagem desejada!

Detetar Porta

porta :: Jogo -> (Int, Int) #

Esta é a função que obtém as coordenadas da Porta para poder, durante a reageEvento avaliar se é o fim do nível ou não. Utiliza findPorta como auxiliar.

findPorta :: Mapa -> Int -> Int -> (Int, Int) #

findPorta irá ler as listas até encontrar a porta e assim devolver as coordenadas corretas. Isto é, se a cada elemnto de uma lista não encontrar, irá adicionar 1 a x, se no fim da lista não tiver encontrado e for para a seguinte, então aumenta 1 a y.
